Request Header Parser Microservice

A freeCodeCamp API project

User story:
  • I can get the IP address, language and operating system for my browser.

Example usage:

https://fcc-header-parser-md.glitch.me/whoami

Example output:

{ "ip_address": "192.168.1.1", "language": "en-US", "software": "Macintosh; Intel Mac OS X 10_11_6" }
